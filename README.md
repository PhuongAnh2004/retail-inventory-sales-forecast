# Supply Chain Data Analysis Project

## Files
- `Retail_inventory.xlsx`: 
  - Source data
  - Analysis dashboards
  - Model results and interpretations
- `Analysis.ipynb`: Python code for data analysis and model building
- `cleaning.py`: Data cleaning functions

## Setup instructions
1. Keep all files in the same folder
2. Required Python packages:
   - pandas
   - numpy
   - matplotlib
   - seaborn
   - statsmodels
   - openpyxl

## Project overview
This project analyzes retail inventory data to develop forecasting models and optimize inventory management. It includes data cleaning, exploratory analysis, and time series forecasting to predict sales and improve stock planning. Key findings highlight seasonal trends, the impact of promotions, and opportunities for better inventory alignment with demand.
